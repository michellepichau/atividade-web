# Atividade Web 🌐

Projetos e exercícios práticos de desenvolvimento web front-end. Desenvolvido durante estudos de HTML e CSS.

## 📋 Sobre

Repositório com atividades práticas de desenvolvimento web, cobrindo estruturação de páginas com HTML5 e estilização com CSS3.

## 🛠️ Tecnologias

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

## 📁 Estrutura

```
atividade-web/
├── index.html        # Página principal
├── style.css         # Estilos
└── assets/           # Imagens e recursos
```

## 🚀 Como visualizar

```bash
# Clone o repositório
git clone https://github.com/michellepichau/atividade-web.git

# Abra o arquivo index.html no navegador
# Ou use a extensão Live Server no VS Code
```

## 📚 Conceitos praticados

- Estrutura semântica HTML5 (`header`, `main`, `footer`, `section`)
- Seletores e propriedades CSS3
- Flexbox e posicionamento
- Boas práticas de organização de código
